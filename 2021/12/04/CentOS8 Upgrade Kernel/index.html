CentOS8 Upgrade Kernel

Posted on 2021-12-03 Edited on 2021-12-04

十五步大法

升级内核

  1. yum upgrade
  2. tar -cvf xxxx.tar.gz -C /usr/src/kernels
  3. yum groupinstall “Development Tools”
  4. yum install ncurses-devel
  5. yum install hmaccalc zlib-devel binutils-devel elfutils-libelf-devel
  6. cp /boot/config-4.18.0-305.3.1.el8.x86_64 .config
  7. sh -c ‘yes “” | make oldconfig’
  8. make clean
  9. make -j56 bzImage
  10. yum install openssl-devel
  11. 打开.config

注释以下三行
CONFIG_MODULE_SIG_KEY=”certs/signing_key.pem”
CONFIG_SYSTEM_TRUSTED_KEYRING=y
CONFIG_SYSTEM_TRUSTED_KEYS=””

修改以下一行
CONFIG_DEBUG_INFO_BTF=n

  1. make -j56 modules
  2. make -j56 modules_install
  3. make
  4. make install
